Gabinator


Gabinator es un proyecto de Screen Sharing entre un dispositivo Windows o Linux como servidor y un dispositivo Android como cliente

Gabinator Android

El cliente del proyecto gabinator, interpreta los datos enviados de Gabinator Desktop

Gabinator Desktop
